Este proyecto consiste en el desarrollo de una aplicación web interactiva utilizando **Streamlit**.

## Descripción
La aplicación permite realizar un análisis exploratorio básico de un conjunto de datos de anuncios de venta de coches en Estados Unidos.

## Funcionalidades
- Visualización de la distribución del kilometraje mediante un histograma.
- Análisis de la relación entre el precio y el kilometraje con un gráfico de dispersión.
- Interacción mediante casillas de verificación.

## Tecnologías utilizadas
- Python
- Pandas
- Plotly Express
- Streamlit

## Enlace a la aplicación
La aplicación está desplegada en Render y es accesible desde el navegador web. https://proyecto-sprint-7-mcp3.onrender.com
